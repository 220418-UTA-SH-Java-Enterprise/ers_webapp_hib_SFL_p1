# PROJECT NAME
Employee Reimbursment System Web App 
## Project Description

A very simple CRUD web application to mantain the growth of reimbursing employees for expneses while hired with a company. 

## Technologies Used

* Java 8 
* Java EE Servlet API 
* JUIT
* Mockito
* Apache Maven 
* Hibernate
* PostGreSQL deployed on AWS RDS
